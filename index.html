My Achievements

NASAMINDS

This is a rocket I programmed for the NASAMINDS compitation with the microcontroller Raspberry PICO connected to a Raspberry Pi which is accessed with a computer via SSH. How it works is I programmed the Raspberry PICO to recive serial input from the Raspberry Pi 3 which is a command that will instuct the PICO to burn a fuse connected to the rocket which will light it to send it off to space. This project won as a finalize for NASA in the NASAMINDS compitation.

Tetris

This is a Tetris game I made with an SoC on the FPGA Nexys A7-100T. I have programmed a song that gets higher pitch depending on what score you have and 60 individual controlable blocks that make the shapes you can rotate. I did this project for my final project for my advanced Verilog course and I enjoyed making it.
